
   LIGHTSMARK 2007                                           v1.0, 2007-10-24
   __________________________________________________________________________


   Lightsmark measures computer performance with next generation 3D engine.


1. Benchmark features

   - realtime global illumination
   - realtime penumbra shadows
   - realtime color bleeding
   - infinite light bounces
   - fully dynamic HDR lighting
   - 220000 triangles
   - 200-300 average fps (GF8800)


2. Supported operating systems

   - Windows XP
   - Windows XP x64
   - Windows Vista
   - Windows Vista x64


3. Supported GPUs

   - NVIDIA GeForce 6xxx
   - NVIDIA GeForce 7xxx
   - NVIDIA GeForce 8xxx
   - AMD (ATI) Radeon 9500-9800
   - AMD (ATI) Radeon Xxxx
   - AMD (ATI) Radeon X1xxx
   - AMD (ATI) Radeon HD2xxx

   Very important! Use the most recent drivers from
   - http://www.nvidia.com/Download/index.aspx
   - http://ati.amd.com/support/driver.html

   Known remaining issues with drivers (request fix from driver vendors):
   - disappearing objects on Radeon HD
   - crash on some Quadro cards
